# Prompts utilizados para primer ejercicio:

## Primero
Genera un prompt con estructura de élite que me ayude a resolver un ejercicio que me han dejado en un programa para desarrolladores de software. Yo te daré el ejercicio y los archivos a considerar y seguirás las instrucciones del ejercicio que te dé. Antes de que generes el prompt, responde, puedes recibir un repositorio de GitHub a través del link para que entiendas el contexto completamente?

## Segundo
Dame el prompt estructurado que considere que te daré el ejercicio y el link del repositorio de GitHub

## Tercero
Ejecuta el prompt y considera lo siguiente: Repositorio: https://github.com/Julio-Glez/AI4Devs-intro-2602-ro Ejercicio: Las isntrucciones del ejercicio están en el archivos README.md del mismo repositorio

## Cuarto
Sí, analiza todo el repositorio archivo por archivo, dime exactamente qué tengo que implementar solamente par el ejercicio de reverse string para completar el ejercicio y dame la solución lista para entregar

## Quinto
La solución que me diste implementa una página web con JavaScript?

## Sexto
Con el mismo repositorio considera el siguiente ejercicio a realizar: "Crea una página web con lógica en javascript que invierta el orden de una cadena de texto. Ejemplo: si introduzco AI4Devs devuelve sveD4IA. Hazlo apoyado en el seed index.html y script.js que proporcionamos dentro de la carpeta template Utiliza un chatbot, como ChatGPT o Gemini, no un asistente de código en IDE como Github Copilot. Para entregar, haz una pull request que incluya una carpeta copiada de template, con el nombre reversestring-iniciales (ejemplo reversestring-ARM). Debe incluir no solo el código generado, sino también, fundamental, el prompt utilizado y el chatbot utilizado en prompts.md. Si has usado más de un prompt hasta llegar a una solución adecuada, añade todos en orden. Añade además el prompt final en el comentario del pull request."
